## Overview

Action Nodes perform deterministic media edits that do not use AI generation. They are instant, free, and predictable -- perfect for resizing, cropping, formatting, and other standard transformations.

<Note>Action Nodes do not consume generation credits since they perform non-AI operations.</Note>

## Action Categories

### Image Actions

Resize, crop, rotate, flip, adjust brightness/contrast/saturation, add borders, convert formats, and apply standard filters.

### Video Actions

Trim, loop, reverse, change speed, extract frames, resize, mute audio, and concatenate clips.

### Text Actions

Change case, find and replace, split, join, format as markdown or JSON, count words, and extract patterns with regex.

## Using Action Nodes

<Steps>
  <Step title="Spawn">Open the Command Palette and search for the specific action (e.g., "Resize Image").</Step>
  <Step title="Connect">Wire an upstream node's output into the Action Node's input.</Step>
  <Step title="Configure">Set parameters like target dimensions, crop region, or text pattern.</Step>
  <Step title="Run">Action Nodes execute instantly and pass the result downstream.</Step>
</Steps>

<Tip>Chain multiple Action Nodes for complex edit pipelines -- e.g., crop, then resize, then add a border -- all without using any credits.</Tip>
