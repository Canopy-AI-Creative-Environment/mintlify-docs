## Overview

The Image Node lets you generate visuals from scratch or transform existing images using state-of-the-art AI models. It supports Elements for character and style consistency.

## Modes

### Text-to-Image

Write a text prompt describing the image you want. Adjust resolution, aspect ratio, and model settings to fine-tune results. Attach an [Element](/editor/elements) to maintain visual identity across generations.

### Image-to-Image

Connect an upstream Image Node or upload a reference image. The model will transform it based on your prompt while preserving the structure and composition of the original. Control the transformation strength with the **influence** slider.

<Tip>Use a low influence value (0.2-0.4) for subtle style changes and a high value (0.7-0.9) for dramatic transformations.</Tip>

## Settings

| Setting | Description |
|---|---|
| **Model** | Select from available image generation models |
| **Resolution** | Output dimensions (up to 2048x2048 on supported models) |
| **Element** | Attach a character, product, or style Element |
| **Seed** | Lock the seed for reproducible results |

<Note>Image generation consumes credits based on resolution and model. Check your usage at [app.trycanopy.space](https://app.trycanopy.space).</Note>
