## What Are Nodes?

Nodes are the building blocks of every Canopy AI workflow. Each node performs a specific function -- generating images, processing text, editing video, or routing data. Connect nodes with **noodles** (the colored lines between ports) to build multi-step AI pipelines.

<Tip>Double-click the canvas or press `Cmd/Ctrl + K` to open the Command Palette and quickly spawn any node type.</Tip>

## How Noodles Work

Each node has **input ports** (left side) and **output ports** (right side). Drag from an output to an input to create a noodle. Data flows left to right -- the output of one node becomes the input of the next. Port colors indicate data type: text, image, video, or audio.

<Note>A single output port can connect to multiple inputs, letting you fan out data to parallel processing branches.</Note>
