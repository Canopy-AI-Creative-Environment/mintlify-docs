## Overview

The Text Node is your all-purpose language processing tool on the Canopy AI canvas. It accepts multiple input types and produces text output, powered by leading LLMs.

## Modes

<CardGroup cols={2}>
  <Card title="Text-to-Text">Write prompts to generate, summarize, translate, or transform text content.</Card>
  <Card title="Image-to-Text">Feed an image into the Text Node to get descriptions, OCR, analysis, or structured data extraction.</Card>
  <Card title="Video-to-Text">Analyze video content to generate scene descriptions, summaries, or transcripts.</Card>
  <Card title="Audio-to-Text">Transcribe audio inputs into text with speaker identification and timestamps.</Card>
</CardGroup>

## Using the Text Node

Connect any upstream node's output to the Text Node's input port. Write your instruction in the prompt field -- use `{{input}}` to reference the incoming data. Select your preferred model from the dropdown and click **Run**.

<Tip>Chain multiple Text Nodes together for multi-step reasoning: summarize first, then translate, then format.</Tip>

## Model Selection

The Text Node supports multiple language models. Choose based on your needs -- faster models for simple tasks, more capable models for complex analysis. Model availability depends on your [Canopy AI plan](https://trycanopy.space/pricing).
