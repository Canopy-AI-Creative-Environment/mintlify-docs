## Overview

The Layer Editor node provides professional image compositing directly on the Canopy AI canvas. Stack multiple images as layers, apply masks, adjust blend modes, and export the composite as a single image for further processing.

## Working with Layers

Connect multiple image sources to the Layer Editor's input ports. Each input becomes a layer that you can reorder, resize, and position. Use the built-in toolbar to adjust opacity, blend modes, and transformations.

<Steps>
  <Step title="Add Layers">Connect image outputs from upstream nodes -- each connection creates a new layer.</Step>
  <Step title="Arrange">Drag layers in the stack to reorder them. Use the transform handles to position and scale.</Step>
  <Step title="Mask and Blend">Apply masks to control visibility and select blend modes (multiply, screen, overlay, etc.).</Step>
  <Step title="Export">The composited result flows out of the Layer Editor's output port to downstream nodes.</Step>
</Steps>

<Tip>Use the Layer Editor between an Image Node and an Export Node to add text overlays, watermarks, or composite multiple generated images into a single scene.</Tip>
