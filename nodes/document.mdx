## Overview

The Document Node lets you bring external documents into your Canopy AI workflows. Drop a PDF onto the canvas and it becomes a data source that downstream nodes can read, summarize, and transform.

<Tip>Drag and drop a PDF directly onto the canvas to auto-create a Document Node.</Tip>

## How It Works

The Document Node extracts text and images from uploaded PDFs. Connect its output to a Text Node for summarization, a Batch Node for page-by-page processing, or any other node that accepts text input.

## Supported Features

| Feature | Details |
|---|---|
| **Text extraction** | Full text content from all pages |
| **Image extraction** | Embedded images available as separate outputs |
| **Page selection** | Process all pages or specify a range |
| **Multi-document** | Upload multiple PDFs to a single node |

<Note>Large documents (100+ pages) may take a few seconds to parse. The extracted content is cached for fast re-runs.</Note>
