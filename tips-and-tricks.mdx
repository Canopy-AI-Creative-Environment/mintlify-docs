## Keyboard Shortcuts

| Action | Mac | Windows |
|--------|-----|---------|
| Command palette | Cmd K | Ctrl K |
| Open Roots Agent | Cmd / | Ctrl / |
| Fit canvas | F | F |
| Undo/Redo | Cmd Z / Cmd Shift Z | Ctrl Z / Ctrl Shift Z |
| Duplicate node | Cmd D | Ctrl D |
| Pan canvas | Space + drag | Space + drag |

## Workflow Patterns

- **Chain text → image → video** for end-to-end content
- **Use Batch nodes** for parallel processing of multiple inputs
- **Create Elements** for character/product consistency
- **Router node** to fan one source to many downstream nodes
- **Let Roots Agent** build complex workflows from a description
