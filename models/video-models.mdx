FLORA's Video Node lets you generate and transform video directly inside your canvas workflow. You can drive it with a text prompt, an image, or an audio file — producing everything from short social clips to cinematic sequences with synced audio. Video models also support post-production tasks like upscaling, lipsync, background removal, and subtitle rendering.

<Warning>
  Video models require a **Starter plan or above**. They are not available on the Free plan. [Upgrade your plan](https://flora.ai/pricing) to unlock video generation.
</Warning>

## Editing & Reference Models

These models edit existing videos, apply lipsync, or use reference inputs to guide generation.

<AccordionGroup>
  <Accordion title="Lipsync">
    Synchronize mouth movement to an audio track or dialogue file:
    - **Fabric 1.0** — flagship lipsync using image and audio inputs
    - **Lipsync 2 Pro** — lip synchronization for dubbed content and talking-head alignment
    - **Sync 3** — next-gen lipsync with studio-grade capabilities
    - **VEED Lipsync** — generate realistic lipsync from an audio file and a video
  </Accordion>
  <Accordion title="Kling Reference & Edit">
    - **Kling O1 Edit / O3 Pro Edit / O3 Standard Edit** — edit video with text or image guidance
    - **Kling O1 Reference / O3 Pro Reference / O3 Standard Reference** — style transfer from a reference video
    - **Kling O1 References / O3 Pro References / O3 Standard References** — keep subjects consistent across scenes using multiple references
  </Accordion>
  <Accordion title="Motion & Pose Transfer">
    - **Marey Motion Transfer** — apply motion from one video to another (choreography transfer, steadicam paths)
    - **Marey Pose Transfer** — copy body poses to new subjects (stunt-viz, freeze-frame-to-motion)
    - **Animatediff** — reanimate video with new motion for looping GIFs and motion graphics
  </Accordion>
  <Accordion title="Video Editing">
    - **Luma Modify Video** — restyle with adhere, flex, or reimagine modes (day-for-night, live-action to anime)
    - **WAN 2.2 Animate Move** — apply motion to a reference character
    - **WAN 2.2 Animate Replace** — swap subjects using a reference image
    - **Grok Imagine References** — generate video guided by one or more reference images
    - **Grok Imagine Edit** — edit a video using text prompts
    - **Runway Gen-4.5** — high-fidelity cinematic motion with strong prompt adherence
  </Accordion>
</AccordionGroup>
