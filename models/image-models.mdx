FLORA's Image Node powers two core workflows: **text-to-image** (generate a new image from a prompt) and **image-to-image** (transform, edit, or enhance an existing image). With over 40 image models spanning generation, editing, vector output, and upscaling, you can build everything from rapid concept sketches to production-ready 4K assets — all inside the same canvas.

<Note>
  Image models are available on **all plans, including Free**. No subscription upgrade is required to start generating images in FLORA.
</Note>

## Editing Models

These models edit or transform an existing image rather than generating from scratch. Feed them an output from an upstream Image or Video node to continue refining your asset.

<AccordionGroup>
  <Accordion title="Structure & Composition">
    - **Flux Canny** — edge detection for precise composition control
    - **Flux Depth** — depth maps for 3D spatial consistency and accurate perspectives
    - **Flux Kontext Max** — outpainting, style swaps, text edits, and character continuity
  </Accordion>
  <Accordion title="Face & Portrait">
    - **Ideogram Character** — transfer a face between images while maintaining character identity
    - **Enhancor V1** — subtle portrait retouching and skin smoothing
    - **Enhancor V3** — professional-grade face enhancement
    - **Enhancor V4** — advanced face enhancement with the latest capabilities
  </Accordion>
  <Accordion title="Text in Images">
    - **Qwen Image Edit** — best for adding or changing text elements within an image
    - **Qwen Image Edit Plus** — complex layouts with multiple text elements
  </Accordion>
  <Accordion title="Isolation & Cleanup">
    - **Remove Background** — isolate clean subjects for transparent backgrounds and compositing
  </Accordion>
</AccordionGroup>

## Vector & SVG Models

Use these models when your output needs to be a scalable vector file rather than a raster image — ideal for logos, icons, and illustration systems.

- **Arrow 1.0** — convert raster images to production-ready SVG vectors
- **Arrow 1.1 Max** — premium vector generation for detailed and complex illustrations; also accepts up to 4 reference images for guided SVG creation
- **Recraft V4 Pro Vector** — high-quality vector illustration generation for brand and design work
- **Recraft V4 Vector** — detailed vector illustrations and graphic design

## Upscaling Models

These models increase resolution or enhance image quality on an existing asset. Connect them downstream of any generation or editing node to finalize your output.

| Model | Scale | Best For |
|---|---|---|
| Magnific Creative Upscaler | 2×–16× (max 10,056px) | Artistic upscaling with stylistic enhancement |
| Magnific Precision Upscaler | 2× | Sharp, faithful upscaling that preserves original detail |
| Topaz Upscaler | 1×–4× | Industry-standard upscaling for professional workflows |
| Topaz Generative Upscaler | 1×–4× | Reconstructs missing detail, faces, and textures |
| Bria Video Upscaler | 2×/4× | Commercial video upscaling with licensing compliance |
