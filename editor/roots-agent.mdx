Roots Agent is an AI agent that lives directly inside the Canopy AI canvas. Describe what you want in plain language and it does the work — reads your canvas, adds and connects nodes, changes models, runs generations, and organizes output. It's not a chatbot — it has hands.

<Info>
  Roots Agent is **free on every plan** and never counts against your usage budget. The only costs are for generation nodes it proposes to run — which you approve first in Assist mode.
</Info>

## Two Operating Modes

<CardGroup cols={2}>
  <Card title="Assist Mode" icon="hand">**Default.** Shows planned nodes and estimated cost, waits for approval.</Card>
  <Card title="Auto Mode" icon="bolt">Runs generations immediately. Maximum speed.</Card>
</CardGroup>

## What Roots Agent Can Do

- Build complete workflows from a description
- Add, modify, or remove nodes; change models, labels, prompts
- Execute image, video, and text nodes — up to **50 at a time**
- Operate Layer Editor tools directly
- Accept image uploads via drag-and-drop (JPEG, PNG, WebP, GIF — up to 5 per message, 20 MB each)

## Providing Context

1. **Select nodes** on canvas before typing — automatically included
2. Type **@** to mention specific nodes by name
3. **Upload images** as reference material

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| Cmd / / Ctrl / | Toggle sidebar |
| Cmd \\ / Ctrl \\ | Dock or float |
| Cmd Enter / Ctrl Enter | Approve pending execution |
| Escape | Cancel or close |

## Memory & Sessions

- **Per-project** — each project has its own session
- **Persistent** — messages saved automatically
- **Session history** — access previous chats from dropdown

<Warning>
  Single-project scope only — cannot access other projects, workspaces, or external websites.
</Warning>
