## Real-Time Collaboration

Invite team members to your project and edit the canvas simultaneously. Each collaborator gets a colored cursor, and all node changes sync in real time. Open the **Share** menu in the top-right corner to manage access.

<Tip>Use `@mentions` in node comments to notify specific teammates about changes or requests.</Tip>

## Sharing and Permissions

| Permission | Can View | Can Edit | Can Publish |
|---|---|---|---|
| **Viewer** | Yes | No | No |
| **Editor** | Yes | Yes | No |
| **Admin** | Yes | Yes | Yes |

Share a project via link or email. Viewers can inspect the workflow and download outputs but cannot modify nodes or connections.

## Publishing to Community

Turn any project into a **published Technique** that other Canopy AI users can discover, fork, and run. Click **Publish to Community** from the Share menu, add a description and tags, and your workflow becomes available at [app.trycanopy.space](https://app.trycanopy.space).

<Note>Published Techniques do not expose your API keys or private Element references.</Note>
