## Navigating the Canvas

The Canopy AI canvas is an infinite 2D workspace where you build AI workflows visually. Pan by clicking and dragging the background, zoom with your scroll wheel or trackpad pinch, and use `Cmd/Ctrl + 0` to reset the view. Double-click empty space to open the **Command Palette** and spawn any node type instantly.

<Tip>Hold `Space` and drag to pan without deselecting nodes -- useful when working in tight layouts.</Tip>

## Spawning and Connecting Nodes

Drag from any node's output port to create a **noodle** -- the colored connection line that carries data between nodes. Drop the noodle onto another node's input port to complete the connection. You can also right-click the canvas to access the full node menu organized by category.

### Keyboard Shortcuts

| Shortcut | Action |
|---|---|
| `Cmd/Ctrl + K` | Open Command Palette |
| `Delete / Backspace` | Remove selected nodes |
| `Cmd/Ctrl + D` | Duplicate selection |
| `Cmd/Ctrl + Z` | Undo |

## Inpainting and Outpainting

Select an Image Node output and use the **brush tool** to mask regions for inpainting -- regenerating only the masked area while preserving the rest. Switch to **outpainting mode** to extend an image beyond its original boundaries, letting the AI fill in new content seamlessly.

<Note>Inpainting and outpainting use additional generation credits based on the output resolution.</Note>
