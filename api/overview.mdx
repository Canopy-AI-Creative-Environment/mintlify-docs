## Overview

The Canopy AI API lets you run any Technique or generation workflow programmatically. Build AI-powered features into your own apps, automate content pipelines, or integrate Canopy AI into your existing toolchain.

<Note>API access requires a **Starter plan** or above. Upgrade at [app.trycanopy.space](https://app.trycanopy.space).</Note>

## Getting Started

<Steps>
  <Step title="Get Your API Key">Generate an API key from **Workspace Settings > API** in your Canopy AI dashboard.</Step>
  <Step title="Read the Docs">Full API reference and guides are available at [developer.trycanopy.space](https://developer.trycanopy.space).</Step>
  <Step title="Make Your First Call">Use the API to run a Technique or generate media -- see [Authentication](/api/authentication) for setup.</Step>
</Steps>

## What You Can Do

<CardGroup cols={2}>
  <Card title="Run Techniques">Execute any Technique via API with custom inputs and receive outputs programmatically.</Card>
  <Card title="Generate Media">Create images, video, audio, and text using the same models available on the canvas.</Card>
  <Card title="Manage Projects">Create, list, and manage projects and their contents via API.</Card>
  <Card title="MCP Integration">Connect Canopy AI to Claude, Cursor, and other AI tools via MCP -- see [MCP](/api/mcp).</Card>
</CardGroup>
