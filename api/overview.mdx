The FLORA API is a REST API that gives you programmatic access to FLORA's AI workflow capabilities. Whether you want to trigger generations from your own backend, retrieve history for your users, or embed FLORA into a larger product, the API is the foundation for building on top of FLORA outside the canvas. The API is out of public beta and available on the Starter plan and above. Full developer documentation — including the complete endpoint reference, request/response schemas, and SDK guides — lives at [developer.flora.ai](https://developer.flora.ai).

<Warning>
  The FLORA API is not available on the Free plan. You must be on a **Starter plan or above** to generate an API key and make requests.
</Warning>

## What you can do with the API

The FLORA API gives you direct, scriptable access to the same capabilities you use in the canvas:

- **Trigger workflow executions** — kick off any workflow programmatically and receive the output in your application
- **Access generation history** — retrieve past generations for a user or workflow to power dashboards, logs, or downstream processing
- **Integrate FLORA into your own applications** — embed FLORA's AI outputs into your product, pipeline, or automation without requiring users to visit the canvas

## Getting started

<Steps>
  <Step title="Upgrade to Starter plan or above">
    The API is not available on the Free plan. Visit your account's billing settings to upgrade to a Starter plan or higher before proceeding.
  </Step>
  <Step title="Get your API key">
    Once you're on an eligible plan, navigate to **Settings → API** in the FLORA app. From there you can generate a new API key or copy an existing one.
  </Step>
  <Step title="Read the full API docs">
    Head to [developer.flora.ai](https://developer.flora.ai) for the complete API reference, including all available endpoints, authentication details, request examples, and SDK documentation.
  </Step>
</Steps>
