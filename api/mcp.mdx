## What Is MCP?

The Model Context Protocol (MCP) is an open standard that lets AI tools like Claude and Cursor connect to external services. Canopy AI's MCP integration lets you trigger Techniques, generate media, and access your workspace directly from your AI coding assistant.

<Tip>MCP turns Canopy AI into a tool that Claude and Cursor can call autonomously during conversations and coding sessions.</Tip>

## Setup

<Steps>
  <Step title="Generate an API Key">Create an API key from **Workspace Settings > API** at [app.trycanopy.space](https://app.trycanopy.space).</Step>
  <Step title="Configure Your Client">Add Canopy AI as an MCP server in your Claude or Cursor configuration file. See the connection details at [developer.trycanopy.space](https://developer.trycanopy.space).</Step>
  <Step title="Use It">Ask Claude or Cursor to generate images, run Techniques, or create content -- it will call Canopy AI automatically.</Step>
</Steps>

## Available MCP Tools

| Tool | Description |
|---|---|
| **run_technique** | Execute a saved Technique with custom inputs |
| **generate_image** | Generate an image from a text prompt |
| **generate_video** | Generate a video from text or image |
| **list_techniques** | Browse your available Techniques |
| **list_projects** | List workspace projects |

## Example Usage in Claude

Once configured, you can ask Claude naturally:

> "Generate a product photo of a coffee mug on a wooden table using Canopy AI"

Claude will call the Canopy AI MCP server, run the generation, and return the result.

<Note>MCP access requires a **Starter plan** or above and a valid `CANOPY_API_KEY`. See [Authentication](/api/authentication) for setup.</Note>
