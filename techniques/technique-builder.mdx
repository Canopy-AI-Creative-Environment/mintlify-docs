## Building a Technique

Any canvas workflow can become a Technique. Design your node graph, test it with sample inputs, then save it as a reusable template.

<Steps>
  <Step title="Design the Workflow">Build your node graph on the canvas as usual -- connect nodes, write prompts, configure settings.</Step>
  <Step title="Mark Inputs">Right-click nodes to mark fields as **Technique Inputs** -- these become the configurable parameters when others run your Technique.</Step>
  <Step title="Test">Run the workflow end to end to verify it produces the expected outputs.</Step>
  <Step title="Save as Technique">Click **Save as Technique** from the canvas menu. Add a title, description, category, and cover image.</Step>
</Steps>

## Publishing

Choose to keep your Technique **private** (workspace only) or **publish to community**. Published Techniques are reviewed and made available to all Canopy AI users.

<Tip>Add clear input labels and a detailed description to help others understand how to use your Technique effectively.</Tip>

## Best Practices

- Keep Techniques focused on a single task for maximum reusability
- Use Elements for visual consistency instead of hardcoding reference images
- Include an Export Node so outputs are easy to download
- Test with varied inputs before publishing

<Note>You can update a published Technique at any time. Existing users will see the updated version on their next run.</Note>
