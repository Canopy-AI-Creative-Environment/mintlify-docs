## Core Concepts

<AccordionGroup>
  <Accordion title="Nodes" icon="cube" defaultOpen>
    The building blocks of every workflow. Each node is a single operation — a model, a text input, an image source, a prompt template, or a utility.
  </Accordion>
  <Accordion title="Noodles" icon="link">
    Connections between nodes. Drag from an output handle to an input to create a live data link that passes results downstream.
  </Accordion>
  <Accordion title="The Canvas" icon="expand">
    Your infinite workspace. Zoom in for detail, zoom out for the big picture, pan freely.
  </Accordion>
</AccordionGroup>

## Get Started

<Steps>
  <Step title="Create your account">Sign up at [app.trycanopy.space](https://app.trycanopy.space) with Google or email.</Step>
  <Step title="Open the canvas">Create a new project and explore the infinite workspace.</Step>
  <Step title="Place your first node">Double-click the canvas or use Cmd K / Ctrl K to spawn a node.</Step>
  <Step title="Connect and generate">Wire nodes together with noodles and hit Run.</Step>
</Steps>
