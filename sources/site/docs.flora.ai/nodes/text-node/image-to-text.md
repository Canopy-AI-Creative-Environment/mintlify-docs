## 

Summary

By connecting an image node to a text node, you can extract information from visual content, style, and composition.

You can also get creative in your ask to abstract the visual information from the image node.

Prompt

- "Describe this image in a few sentences",
 
- "What hidden rhythm flows through this scene?",
 
- "Give me a visual analysis of the composition of this image",
 
- "Conjure a visual poem for this".
