Markdown versions of documentation pages are available by appending \`.md\` to page URLs; this page is available as \[Markdown\](https://docs.flora.ai/plans-and-billing/model-pricing.md). # Model Pricing Per-generation cost for every model FLORA supports. On \*\*Starter and above\*\*, each generation deducts its dollar cost from your workspace's monthly usage budget — see \[How Pricing Works\](/plans-and-billing/pricing.md) for plan budgets, or the \[Pricing FAQ\](/plans-and-billing/faq.md) for billing details. The \*\*Free\*\* plan uses \*\*limited complimentary generations\*\* instead of a monthly dollar pool (see \[flora.ai/pricing\](https://flora.ai/pricing)). \* \*\*Failed generations don't count.\*\* \* \*\*FAUNA is free\*\* on every plan and doesn't draw from your budget. \* \*\*On-demand spending uses the same rates\*\* — no markup for going over your monthly budget. {% hint style="success" %} \*\*Launch bonus extended through August 1, 2026.\*\* Every paid plan has extra usage on top of its baseline. See the \[Launch Bonus\](/plans-and-billing/launch-bonus.md) page. {% endhint %} ## Reading the usage meter Every model has a usage meter — a small row of flowers indicating roughly what a generation will cost. More flowers, higher cost. When the cost band is \*\*$2 or more\*\*, the meter shows \*\*three flowers plus a “+”\*\*.

\### Show exact dollar amounts If you'd rather see the precise cost than read the flower meter, turn on \*\*Preferences → Show generation costs in dollars\*\*. Open Preferences from the workspace menu — the toggle is global and applies to every project in your account.

![Preferences panel with Show generation costs in dollars toggled on](https://docs.flora.ai/files/mgLlTmdb8xEr3Lo8uHwa)

Preferences → Show generation costs in dollars

With it enabled: \* Every model in the picker shows its per-generation cost (e.g. \`$0.052\`) in place of the flower meter.

![Model list showing FREE and dollar costs next to each model](https://docs.flora.ai/files/eyGKt8DqdBOKiCeJYqd4)

Per-model cost shown in the picker

\* The \*\*Run\*\* button on each node shows the exact dollar amount for the run, accounting for batch count and any per-run modifiers.

![Run button showing the exact dollar cost for the current run](https://docs.flora.ai/files/4H26YaphZqshwDB3YXKE)

Run button with the exact cost for the current run

For the exact rate per model, see the tables below. {% hint style="info" %} \*\*Pricing is dynamic — your cost changes with the parameters you choose\*\*, like resolution and clip length. For Image and Video models, the \*\*Cost per generation\*\* column is the price at each model's \*\*Default settings\*\*, and \*\*Price range\*\* shows how low and high the cost can go as you change those settings (each end notes the settings that produce it). For the price of every individual combination, see \[Pricing Breakdown by Parameter\](/plans-and-billing/parameter-pricing.md). \*\*The exact cost for your specific parameters is always shown in the app before you generate.\*\* {% endhint %}
