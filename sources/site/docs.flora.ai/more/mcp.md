The FLORA MCP server is out of public beta. For full documentation on using FLORA outside the canvas via MCP, visit the [FLORA Developer Site](https://developer.flora.ai/mcp).

The legacy `/mcp` beta endpoint has been deprecated. If you previously connected via the beta endpoint, update your MCP client configuration to use the `/api/v1`\-backed install URL documented on the Developer Site.
