**Overview**

Updated BFL model with high customizability and acceleration options.

### 

**Quick facts**

- **Modes:** Text → Image · Image → Image (editing)
 
- **Default output / size:** 1024×1024 (1:1)
 
- **Aspect ratios:** Multiple standard ratios supported
 

---

### 

**What it's great for**

- Fast, customizable image generation.
 
- Acceleration options for speed optimization.
 
- Cost-effective high-quality outputs.
 
- Both generation and editing workflows.
