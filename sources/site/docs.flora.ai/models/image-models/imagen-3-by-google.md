**Overview**

High-quality image editing model by Google DeepMind.

### 

**Quick facts**

- **Modes:** Image → Image (editing)
 
- **Default output / size:** Matches input or 1024×1024
 
- **Aspect ratios:** Auto, 1:1, 3:4, 4:3, 9:16, 16:9
