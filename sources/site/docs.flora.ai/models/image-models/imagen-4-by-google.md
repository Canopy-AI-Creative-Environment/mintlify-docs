**Overview**

Advanced image generation model by Google DeepMind with high-quality outputs.

### 

**Quick facts**

- **Modes:** Text → Image
 
- **Default output / size:** 1024×1024 (1:1)
 
- **Aspect ratios:** 1:1, 3:4, 4:3, 9:16, 16:9

### 

**What it's great for**

- High-quality image generation with Google's latest AI.
 
- Photorealistic outputs with strong detail.
 
- Versatile aspect ratios for different use cases.
 
- Professional-grade image creation.
